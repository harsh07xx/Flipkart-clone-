# Flipkart Clone

A single-file front-end clone of the Flipkart e-commerce homepage, built with plain HTML, CSS, and JavaScript — no frameworks or build tools required.

## Features

- Product grid with cards, pricing, discount badges, and wishlist toggle
- Responsive layout
- Sticky navigation bar with search

## Getting Started

No installation needed. Just open `index.html` in any modern web browser:

```bash
# Clone the repo
git clone https://github.com/<your-username>/flipkart-clone.git
cd flipkart-clone

# Open index.html directly, or serve it locally:
python3 -m http.server 8000
```

Then visit `http://localhost:8000` in your browser.

## Live Demo (GitHub Pages)

If GitHub Pages is enabled on this repo (Settings → Pages → Branch: `main`, folder: `/root`), it will be live at:

```
https://<your-username>.github.io/flipkart-clone/
```
